# 🔢 Binary ↔ Decimal Converter

A simple and responsive web application that converts:

* Binary numbers to Decimal numbers
* Decimal numbers to Binary numbers

Built using HTML, CSS, and JavaScript.

# ⚡ How It Works

### Binary to Decimal

The user enters a binary number, and JavaScript converts it into a decimal number.

### Decimal to Binary

The user enters a decimal number, and JavaScript converts it into binary format.
